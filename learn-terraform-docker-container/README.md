# terraform on AWS

## Getting started

https://developer.hashicorp.com/terraform/tutorials/aws-get-started/aws-build

1. install TF cli

- install-terraform
https://developer.hashicorp.com/terraform/install#linux

2. RUN

 ```console
 terraform init
 terraform fmt
 terraform plan
 terraform apply
 ```

Or apply with direct approval using  terraform apply -auto-approve
